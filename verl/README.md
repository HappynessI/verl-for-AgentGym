# Modified verl Components

This directory contains the modified `verl` training stack used by the Prefix-GRPO experiments.
It is included to make the review artifact self-contained.

For paper review, start from the repository root `README.md` and `ARTIFACT.md`. The most relevant implementation files are the actor loss and trainer code under `verl/verl/`, while data and metrics are indexed under `data/` and `results/`.

Generated checkpoints, local logs, and experiment tracker state are intentionally not included.
