# Results

This directory contains curated training metrics for the Prefix-GRPO review
artifact.

Training metrics:

```text
training_metrics_index.csv
paper_metrics.csv
training/textcraft/
training/babyai/
training/alfworld/
```

`training_metrics_index.csv` summarizes all included metric files and provides
SHA256 hashes. `paper_metrics.csv` is the curated mapping from paper-facing
experiment groups to final metric rows.
