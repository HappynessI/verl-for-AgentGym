This directory contains the recovered ALFWorld Prefix-GRPO training metric file:

```text
training_metrics_prefix_grpo.csv
```

Rows: 1120

logs. Raw local logs are not included because they may contain run-time machine
paths and process details.

`actor_prefix_token_count`, and `actor_prefix_advantage_abs_mean`.
